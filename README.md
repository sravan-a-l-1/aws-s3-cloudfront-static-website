# AWS Static Website Hosting with CloudFront

## Overview
This project demonstrates hosting a static website on Amazon S3 and accelerating global content delivery using Amazon CloudFront CDN.

## AWS Services Used
- Amazon S3
- Amazon CloudFront
- IAM

## Architecture
User → CloudFront → S3 (Static Website Hosting)

## Steps Performed
- Created an S3 bucket and enabled static website hosting
- Configured bucket policy for public access
- Set up CloudFront distribution with HTTPS redirection
- Configured default root object as index.html

## What I Learned
- Static website hosting on AWS
- CDN and caching concepts
- Public access management using bucket policies
- Basic cloud architecture design
